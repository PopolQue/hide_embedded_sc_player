SCPlayer

A reusable, themeable SoundCloud player component for festivals, labels, and radio stations. This demo uses a green accent theme to show how easily colors can be customized.

React Component

Install via npm and use in any React app with full TypeScript support.

Themeable

Every color and dimension is customizable via CSS custom properties.

Persistent Playback

Use the shell pattern for audio that survives page navigation.

Accessible

ARIA labels, roles, keyboard navigation, and screen-reader support.
